Git Quiz

What is Git and why is it so important?

Git is a version control system which allows collaboration between developers.

What’s a branch and why do we create them?

A branch is a new copy of the project that you can develop before committing to the master branch. Why? Because we can test and trial before making changes to the original project.

What’s the working tree?

The working tree is the folder where the repo is initialised(which includes all the subdirectories and files).

What’s the staging area?

The staging area is the area that holds all the files we’re ready to commit and push to the repository.

What’s the command to retrieve the latest version of your files from the remote serve?

git pull origin main
